Alice Tekniska Specifikationer

Arkitektur

  • Backend: FastAPI med Python 3.9+
  • Frontend: Next.js 15.4.6 med React
  • Database: SQLite med FTS5 för full-text search
  • AI Engine: Local gpt-oss:20B via Ollama

Minnessystem

Alice använder ett RAG (Retrieval Augmented Generation) system med:

  1. SQLite database för strukturerat minne
  2. OpenAI embeddings för semantisk sökning
  3. BM25 + embedding hybrid retrieval
  4. Automatic text chunking för stora dokument
